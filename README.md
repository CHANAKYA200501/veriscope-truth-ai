#  VeriScope Truth AI

AI-Powered Multimodal Misinformation Detection System

VeriScope Truth AI is a full-stack web application designed to detect fake and real digital content such as URLs, text, images, and media. The system analyzes content and provides prediction, confidence score, and trust score to help users verify authenticity and prevent misinformation.

---

#  Features

- ✅ Fake and Real URL Detection
- ✅ Trust Score Calculation
- ✅ Confidence Score
- ✅ Backend API Integration
- ✅ MongoDB Database Storage
- ✅ Real-time Analysis
- ✅ Full Stack Implementation

```bash
